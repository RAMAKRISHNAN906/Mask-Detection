About MaskGuard AI

Learn about our AI-powered face mask detection system

Our Mission

MaskGuard AI is an advanced face mask detection system built to help organizations and communities ensure safety compliance. Using state-of-the-art deep learning and computer vision, our system provides real-time, accurate mask detection across multiple input sources.

Whether monitoring a live webcam feed, analyzing uploaded images, or processing video files, MaskGuard AI delivers instant, reliable results with detailed analytics and reporting capabilities.

Technology Stack

MobileNetV2 Deep Learning

Transfer learning with MobileNetV2 architecture for efficient and accurate mask classification

OpenCV Computer Vision

Advanced face detection using DNN module or Haar Cascades for robust face localization

Flask Web Framework

Lightweight Python backend with RESTful API endpoints for all detection operations

Analytics & Reporting

Chart.js visualizations, SQLite logging, and PDF report generation

Key Features

Real-time webcam mask detection
Multi-face detection support
Image and video file processing
AI confidence scoring
Comprehensive analytics dashboard
PDF report generation
Voice alerts for no-mask detection
Auto screenshot capture
